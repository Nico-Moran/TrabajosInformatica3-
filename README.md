# TrabajosInformatica3-
-----------------------------------------
Repositorio de Trabajos Prácticos en Java
-----------------------------------------
Este repositorio reúne nueve trabajos prácticos desarrollados en Java, enfocados en afianzar los fundamentos de la programación orientada a objetos y el manejo de estructuras de datos.
Cada trabajo aborda un tema puntual, desde los conceptos iniciales hasta la implementación de algoritmos y estructuras avanzadas.
-------------------------
Contenido del repositorio
-------------------------
1. Sistema de Gestión de Tareas
Desarrollo de una aplicación sencilla para organizar y administrar tareas, aplicando clases, métodos y principios básicos de orientación a objetos.

2. Sistema de Gestión de Pizzería
Simulación de un sistema de pedidos para una pizzería, con manejo de clientes, productos y lógica de negocio utilizando estructuras de datos.

3. Clases y Objetos
Ejercicios introductorios centrados en la definición de clases, atributos, constructores y métodos para reforzar el paradigma orientado a objetos.

4. Recursividad
Resolución de problemas clásicos mediante algoritmos recursivos, explorando la llamada a métodos dentro de sí mismos.

5. Pila y Cola con Arreglos
Implementación de estructuras lineales (stack y queue) utilizando arreglos, con operaciones básicas de inserción y eliminación.

6. Lista Enlazada Simple
Desarrollo de una lista enlazada con operaciones para insertar, eliminar y recorrer nodos, demostrando el manejo de referencias dinámicas.

7. Árbol AVL
Construcción de un árbol binario de búsqueda balanceado (AVL), incluyendo rotaciones y control automático del equilibrio.

8. Árbol Rojo-Negro
Implementación de otra variante de árbol balanceado, analizando sus propiedades, inserciones y procesos de reequilibrio.

9. Montículo Binario (Heap)
Creación de un montículo binario para gestionar prioridades, con funciones de inserción y extracción ordenadas según prioridad.
----------------------------------
Tecnologías y conceptos aplicados
----------------------------------
Lenguaje Java 8 o superior

Programación Orientada a Objetos (POO)

Estructuras de datos: arreglos, listas, pilas, colas, árboles

Algoritmos recursivos y principios de abstracción

Enfoque modular con proyectos independientes por práctica
